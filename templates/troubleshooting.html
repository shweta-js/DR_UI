Common Issues and Solutions



  • Issue: Poor Internet Connection

    Solution: Ensure that you have a stable internet connection. Try refreshing the page or restarting your router.

  • Issue: Unsupported Image Format

    Solution: Verify that you are uploading or capturing images in supported formats (e.g., JPEG, PNG).

  • Issue: Blurry or Low-Resolution Images

    Solution: Use clear and high-quality images for better prediction accuracy.

  • Issue: File Size Limitations

    Solution: Check the maximum file size allowed for image uploads. Compress or resize images if they exceed the limit.

  • Issue: Webcam Access

    Solution: Grant necessary permissions to access the webcam. Check your browser settings and enable webcam access if required.

  • Issue: Browser Compatibility

    Solution: Ensure that you are using a compatible web browser. Try accessing the website using a different browser if issues persist.

  • Issue: Clear Browser Cache

    Solution: Clear your browser cache and cookies, which can sometimes cause conflicts or interfere with website functionality.

  • Issue: Unexpected Behavior or Errors

    Solution: Refresh the page to resolve temporary issues or glitches. Contact our support team if problems persist.

If you continue to experience difficulties or have any other questions, please contact our support team for further assistance.
